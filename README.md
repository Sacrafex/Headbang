# Headbang
A language designed to make you suffer.

Only 4 symbols allowed: !, &, #, >
Every 4-character chunk represents one character in JS
No newlines. No mercy.

For example,

Javascript: ```console.log("Hi");```

Headbang: ```!!!#!!>#!!>&!&!#!!>#!!#>!!&!&!#>!!#>!!>#!!&#&!&&!>>>!#!&!!#!!>>>&!&#&!>#```
